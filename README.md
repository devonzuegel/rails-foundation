Venture Graph
================

Venture Graph shows the network of funding relationships between companies and venture capital firms featured on Crunchbase Daily.

Ruby on Rails
-------------

This application requires:

- Ruby 2.1.5
- Rails 4.2.0

Learn more about [Installing Rails](http://railsapps.github.io/installing-rails.html).
